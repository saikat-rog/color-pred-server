## ğŸ¯ Key Design Decisions

### 1. **Lowest Bet Wins**
- Prevents whales from dominating
- Encourages balanced betting
- More unpredictable outcomes

### 2. **30-Second Lock**
- Prevents last-second manipulation
- Gives system time to calculate
- Creates suspense for users

### 3. **Immediate Balance Debit**
- Prevents double-spending
- Ensures user commitment
- Simplifies accounting

### 4. **1.8x Multiplier (Configurable)**
- House edge built in
- Adjustable for profitability
- Database-driven for flexibility

### 5. **480 Periods/Day**
- 24/7 operation
- High engagement
- Clear period identification

### 6. **Automatic Period Management**
- No manual intervention
- Server restart safe
- Fault tolerant
